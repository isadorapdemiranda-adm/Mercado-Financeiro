#  Brazilian Stocks Financial Analysis - PETR4 (Petrobras)

**Educational project for financial analysis using Python, yfinance, and pandas.**

Complete analysis of **Vale (VALE3.SA)** stock using historical data from 2023 to April 2026.

---

## Project Objective

To develop a **complete and educational financial analysis** of companies listed on the Brazilian Stock Exchange (B3), focusing on:

- Technical Analysis (price, moving averages, volatility)
- Return calculations and performance
- Fundamental Analysis (P/E, Dividend Yield, Market Cap, etc.)
- Professional interactive visualizations

Perfect for students and professionals in **Finance, Economics, Data Science, and Business Administration**.

---

## ✨ Features

- Automatic data download via `yfinance`
- Data cleaning and preprocessing
- Daily and cumulative returns calculation
- Moving Averages (SMA 20, 50, and 200)
- Interactive charts with Plotly
- Candlestick chart
- Volatility analysis
- Consolidated fundamental data
- Modular structure (easy to adapt for other companies)

---

## 🛠️ Technologies Used

- **Python 3**
- `pandas` – Data manipulation
- `yfinance` – Financial data download
- `plotly` – Interactive visualizations
- `matplotlib` + `seaborn` – Static charts
- `openpyxl` + `xlsxwriter` – Excel handling
